# My start-up Demo front-end version
A web platform connecting tourists in Adjara, Georgia with local restaurants, theatres, and events through a QR-based discount system. 
# What it does
Tourists arriving in Batumi receive a unique QR code from their taxi driver or tour guide. After scanning and registering on the platform, they get a 10% discount at all partner venues. The system automatically tracks which agent referred each tourist, enabling transparent commission distribution.
# Features
Tourist & agent registration with separate flows
QR-based referral tracking
Partner venue directory (restaurants, cafés, hotels, theatres)
Events section (unlocked after registration)
7-language support: Georgian, English, Russian, Turkish, Arabic, German, French
